User Guide
==========

Use Moirae in stages:

1. *Select Model* that describes the physics. See `System Models <system-models.html>`_
2. *Initialize parameters* for the model.
   `Extractors <extractors.html>`_ make guesses quickly.
3. *Estimate parameters* using time series data. See `estimators <estimators.html>`_ for
   the available strategies.

The documentation explains each of these steps in sequential order.
