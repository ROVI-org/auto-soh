Moirae
======

Moirae is a collection of tools for estimating how the state of health for a battery evolves with time.
The core features are a set of online and offline algorithms for estimating the performance parameters
which define the physics of a variety of storage systems.

Why the name?
-------------

The `Moirae <https://en.wikipedia.org/wiki/Moirai>`_ are the three personifications of fate in Greek Mythology.
It seemed fitting a name for a library whose purpose is to judge the life of a battery and
the possibility of moving it on to the next.
